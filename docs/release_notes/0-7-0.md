  * Pass 3: Generate JSON only for valid selectors with filtered addOutbounds

- addOutbounds filtering logic:
  * Dynamic selectors (in outboundsInfo) are added only if isValid == true (outboundCount > 0)

  * Empty dynamic selectors are not generated and not added to addOutbounds
